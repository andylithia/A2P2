===== Power Supply Reference Design  Schematics =====

This archive contains two schematics sheets plus postscript files
of each sheet.  The postscript files are formatted for an 8 1/2 x 11
piece of paper.

All schematic files are compatable with Orcad/SDT III Version 3.22.

source.sch ------ AC power input, pre-regulator and battery
cntrl.sch ----- DC-DC Switching converter

The required library to use with the ULP Intel486(tm) SX processor Board
Schematics is:

ps_ref.lib
