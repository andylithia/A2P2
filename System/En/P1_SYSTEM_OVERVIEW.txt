7.1 [Vector Overview]

    +-----------+---------+-------------------------------------------+
    |  Vector   |   In t  | Function                                  |
    |           |   ID    |                                           |
    +-----------+---------+-------------------------------------------+
    |   0 - 3   |    0    | Divide by Zero                            |
    |   4 - 7   |    1    | Single Step                               |
    |   8 - B   |    2    | NMI                                       |
    |   C - F   |    3    | INT3                                      |
    +-----------+---------+-------------------------------------------+
    |  10 - 13  |    4    | Overflow                                  |
    |  14 - 17  |    5    | [COPY] Key                                |
    |  18 - 1B  |    6    | [STOP] Key                                |
    |  1C - 1F  |    7    | Interval Timer                            |
    +-----------+---------+-------------------------------------------+
    |  20 - 23  |    8    | Timer                                     |
    |  24 - 27  |    9    | Keyboard                                  |
    |  28 - 2B  |    A    | CRTV(V-SYNC)                              |
    |  2C - 2F  |    B    | Expansion Bus INT0                        |
    +-----------+---------+-------------------------------------------+
    |  30 - 33  |    C    | RS-232C (ch0)                             |
    |  34 - 37  |    D    | Expansion Bus INT1                        |
    |  38 - 3B  |    E    | Expansion Bus INT2  (CMT)         (NOTE1) |
    |  3C - 3F  |    F    | System Reserved                           |
    +-----------+---------+-------------------------------------------+
    |  40 - 43  |   10    | Printer/NDP                       (NOTE2) |
    |  44 - 47  |   11    | Expansion Bus INT3  (HD)                  |
    |  48 - 4B  |   12    | Expansion Bus INT41 (640KB FD)            |
    |  4C - 4F  |   13    | Expansion Bus INT42 (1MB FD)              |
    +-----------+---------+-------------------------------------------+
    |  50 - 53  |   14    | Expansion Bus INT5  (Sound)               |
    |  54 - 57  |   15    | Expansion Bus INT6  (Mouse)       (NOTE3) |
    |  58 - 5B  |   16    | NDP/Printer                       (NOTE4) |
    |  5C - 5F  |   17    | Noise (System Reserved)           (NOTE6) |
    +-----------+---------+-------------------------------------------+
    |  60 - 63  |   18    | KB/CRT BIOS                               |
    |  64 - 67  |   19    | RS-232C BIOS                              |
    |  68 - 6B  |   1A    | Printer BIOS                              |
    |  6C - 6F  |   1B    | DISK BIOS                                 |
    +-----------+---------+-------------------------------------------+
    |  70 - 73  |   1C    | Calendar Timer BIOS                       |                              |
    |  74 - 77  |   1D    | System Reserved                   (NOTE5) |
    |  78 - 7B  |   1E    | N88-BASIC                                 |
    |  7C - 7F  |   1F    | System Reserved                           |
    +-----------+---------+-------------------------------------------+
    |  80 ~ FF  | 20 ~ 3F | System Reserved                           |
    +-----------+---------+-------------------------------------------+
    | 100 ~ 1FF | 40 ~ 7F | User                                      |
    +-----------+---------+-------------------------------------------+
    Unused vectors are set to dummy addresses.
    Vector 8~17 are occupied by hardware interrupts.

    Note 1: PC-9801 uses INT2 for the CMT interrupt.
            In high-res mode, INT2 is assigned to the mouse interrupt.
    Note 2: When using 8086/70116 CPU, this vector is assigned to printer.
            When using 80286/386/486/Pentium, this vector is assigned to NDP.
    Note 3: In normal mode, the mouse interrupt can be assigned to any from 
              INT0 to INT6 using the strap switch(jumper).
            In high-res mode, the mouse interrupt is fixed to INT2.
    Note 4: In normal mode, when using 80286/70116 CPU, this vector is assigned 
              to NDP, or when using 80286/386/486/Pentium, this vector is not 
              assigned.
            In high-res mode, this vector is assigned to printer.
    Note 5: In high-res mode, this vector is assigned to GRAPH BIOS.
    Note 6: PC-9821Ap, As, Ae, Af, Ne uses system timer.
